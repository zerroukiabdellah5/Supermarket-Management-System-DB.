# Supermarket Inventory & Sales Management (SQL)

## Overview
This project showcases a professional database schema designed for retail environments, specifically supermarkets. It handles inventory management, sales tracking, and reporting using Oracle SQL logic.

## Key Features
- **Relational Structure:** Linked tables for Categories, Products, and Sales.
- **Data Integrity:** Implemented Primary Keys, Foreign Keys, and Constraints.
- **Business Intelligence:** Pre-built queries for inventory alerts and revenue analysis.

## Professional Background
Designed by an IT Specialist with 3+ years of practical experience in supermarket technical systems.
